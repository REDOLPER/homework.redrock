2. 合并边框

使用 border-collapse: collapse; 合并相邻边框。

3. 样式化表格

为表头添加不同颜色，使表格更具视觉吸引力。

部门 负责人 预算
技术部 王经理 ¥500,000

.styled-table { border: 2px solid #3498db; border-collapse: collapse; } .styled-table th { background-color: #3498db; color: white; border: 1px solid #2980b9; padding: 10px; } .styled-table td { border: 1px solid #ddd; padding: 8px; }

4. 圆角表格

使用 border-radius 和 overflow: hidden 创建圆角表格。

课程 教师 时长
网页设计 张老师 36小时
JavaScript 李老师 48小时
.rounded-table { border: 2px solid #e74c3c; border-collapse: separate; border-radius: 10px; overflow: hidden; } .rounded-table th { background-color: #e74c3c; color: white; padding: 10px; } .rounded-table td { border: 1px solid #ddd; padding: 8px; }

5. 简约风格
